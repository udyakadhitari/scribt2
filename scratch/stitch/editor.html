Dijkstra's algorithm is an algorithm for finding the shortest paths between nodes in a graph, which may represent, for example, road networks. It was conceived by computer scientist Edsger W. Dijkstra in 1956 and published three years later.

info Type '/' for commands

1. Core Concepts

  • Graph Representation: We typically use an adjacency list for better performance on sparse graphs.
  • Priority Queue: Essential for efficiently retrieving the next node with the minimum tentative distance.
  • Relaxation: The process of updating the shortest path estimate to a vertex.

2. Python Implementation

Below is a standard implementation using Python's `heapq` module.

python 

import heapq

def dijkstra(graph, start):
    distances = {node: float('infinity') for node in graph}
    distances[start] = 0
    pq = [(0, start)]
    
    while pq:
        current_distance, current_node = heapq.heappop(pq)
        
        if current_distance > distances[current_node]:
            continue
            
        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight
            
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(pq, (distance, neighbor))
                
    return distances
